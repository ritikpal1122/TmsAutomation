# Insights/Analytics Features

> **Product URL**: https://analytics.lambdatest.com/
> **Nature**: Feature provided to all customers (NOT a separate SKU)
> **Also known as**: Analytics (internal name)

## Overview

Insights is the unified analytics and dashboarding platform for TestMu AI that enables QA managers, executives, and engineers to track tests conducted across all platform products and gain deep-level insights through customizable dashboards and widgets.

**Key Value Proposition**:
- Cross-product analytics in a single view
- AI-powered test intelligence (Smart Tags, Flaky Detection, RCA)
- Customizable dashboards with pre-built and custom widgets
- Real-time data refresh (30 second max latency)
- Completely free for all users (AI RCA credits purchased separately)

**Data Retention**: 2 months by default for all products

## 9. Data Architecture

### 9.1 Data Flow

```
Test Execution Products → Kafka Events → Log-shipper → OpenSearch → LPA (Insights API) → Dashboard
         │                                    │
         ▼                                    ▼
   Emit events on                    Fetch related metadata
   test completion                   from product APIs
```

### 9.2 Architecture Components

| Component | Role |
|-----------|------|
| **Kafka** | Event streaming for test completion events |
| **Log-shipper** | Data ingestion service |
| **OpenSearch** | Dedicated product-level indices |
| **LPA (Insights API)** | Reads from OpenSearch, responds to dashboard queries |

### 9.3 Data Latency

| Metric | Value |
|--------|-------|
| **Dashboard Visibility** | Max 30 seconds from test completion |
| **Unique Test Instances** | Max 5 minutes processing time |
