### Linking Jira Issues in Test Manager[â](https://www.testmuai.com/support/docs/link-jira-issues-with-test-manager#linking-jira-issues-in-test-manager "Direct link to Linking Jira Issues in Test Manager")

Once the integration is complete, follow these steps to link issues with your test cases or test runs:

  1. Head to the issues section in Test Manager, click on **Link Issue** and go to Jira.

> Note: Issue linking is supported in both Test Cases and Test Runs. To link issues, go to the Issues tab within a Test Case or a Test Run.

![Link Azure DevOps Issue](https://www.testmuai.com/support/assets/images/4-03e520c59df7d5e925682fa4f5df6e6d.png)

  2. Enter the Jira Issue ID or URL in the input field and click on **Link Issue**.

![Link Azure DevOps Issue](https://www.testmuai.com/support/assets/images/Jira-linkIssue-enterId-c70494f80bb61ea527fa36aae03f8736.png)

  3. The Jira issue from your integrated Jira Project is successfully linked and listed. You can view the Issue summary, ID, type, status, priority, and creator.

![Link Azure DevOps Issue](https://www.testmuai.com/support/assets/images/Jira-Issue-Listed-a29e8f67e6b489fc68b9a1493665cbc6.png)

  4. To **unlink an issue** , click on the unlink button located on the right side of the Jira Issue.

![Link Azure DevOps Issue](https://www.testmuai.com/support/assets/images/Jira-Unlink-Issue-fc0145f52a919d88df08618ec0b0dee8.png)
